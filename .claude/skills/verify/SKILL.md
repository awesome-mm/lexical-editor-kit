---
name: verify
description: 맨 마지막 코드 수정 이후 pnpm run build (빌드)와 pnpm run dev로 (로컬 서버 가동) 이상이 없는지 확인한다
---

1. 코드에 대한 수정 요청이 있을 경우에만 실행하며 맨 마지막 코드 수정 이후 아래의 사항을 실행하며 필수적으로 점검한다.
2. 생성한 코드가 적절한 코드인지를 재검증해야합니다.
3. pnpm run build 를 통한 코드 수정 후 빌드에 이상이 없는지 확인해야합니다.
4. pnpm run dev를 통한 코드 수정 후 로컬 서버 가동이 이상이 없는지 확인해야합니다.
5. 코드 오류가 있을 경우 오류를 파악하고 코드 수정합니다. 수정 후에는 다시 오류가 없고 정상적인 작동과 요구사항을 만족할때까지 1~4번을 반복합니다.
